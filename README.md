# Memoiré 💫

A cinematic MERN memory journal platform designed to preserve moments beautifully through immersive animations, aesthetic visuals and modern web technologies.

---

## 🌟 Features

- 📸 Upload and store memories dynamically
- ☁️ Cloudinary image hosting
- 🎬 GSAP + Framer Motion animations
- 🌙 Dark / Light mode
- 💖 Floating hearts and cinematic effects
- 🎵 Background music player
- ✨ Custom glowing cursor
- 📱 Fully responsive design
- 🚀 Smooth scrolling experience
- 🎨 Glassmorphism UI design

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Framer Motion
- GSAP

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas

### Media Storage
- Cloudinary
